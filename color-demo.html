Utilisation dans le projet

Les nouvelles couleurs ont été appliquées aux variables CSS suivantes :

  • --main-h: 15 (teinte orange)
  • --main-s: 89% (saturation)
  • --main-l: 54% (luminosité)
  • --primary-500: #f05623
  • --white: #ffffff

Ces changements affectent tous les éléments du site qui utilisent les variables CSS principales.
